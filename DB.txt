Убедитесь, что ваш контейнер PostgreSQL запущен: 

    docker ps

Используйте docker exec для подключения к контейнеру: 

    docker exec -it project-db-1 psql -U hotel_user -d hotel_db

Внутри psql выполните следующие команды: 

    CREATE TABLE rooms (
        id SERIAL PRIMARY KEY,
        number VARCHAR(10) NOT NULL,
        type VARCHAR(50) NOT NULL,
        available BOOLEAN DEFAULT TRUE
    );

Добавьте несколько записей в таблицу:

    INSERT INTO rooms (number, type, available) VALUES
    ('101', 'Single', TRUE),
    ('102', 'Double', FALSE),
    ('103', 'Suite', TRUE);

Убедитесь, что данные добавлены:

    SELECT * FROM rooms;

Запустите все сервисы:

    docker-compose up --build

Перейдите в папку frontend и запустите PyQt6-приложение:

    cd frontend
    pip install -r requirements.txt
    python main.py
